# YouTube Downloader Pro

YouTube Downloader Pro is a desktop application foundation for a professional video download manager built with Python 3.12+ and PySide6.

This repository currently contains the initial clean architecture scaffold for the project. The graphical interface and download workflows are intentionally not implemented in Commit 001, so the codebase can begin with a clear modular structure before application features are added.

## Requirements

- Python 3.12 or newer
- PySide6

## Setup

Create and activate a virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

Install dependencies:

```powershell
pip install -r requirements.txt
```

Run the application entry point:

```powershell
python app.py
```

## Project Status

Version `0.1.0` establishes the repository structure, dependency baseline, documentation, and application entry point. UI implementation will be added in a later milestone.
